# Rust 重实现 N_m3u8-RE 项目计划

## 1. 项目分析

### 1.1 项目结构
- 原项目：C# 实现的 DASH/HLS/MSS 下载器
- 目标：使用 Rust 重新实现，保持相同的命令行参数和功能
- 已有的 Rust 实现结构：
  - `src/main.rs`：主入口
  - `src/commandline/`：命令行参数解析
  - `src/parser/`：媒体流格式解析（HLS、DASH、MSS）
  - `src/downloader/`：下载管理器（点播和直播）
  - `src/crypto/`：加密解密功能
  - `src/muxer/`：媒体文件混流
  - `src/utils/`：工具函数
  - `src/i18n/`：国际化支持
  - `src/entity/`：数据结构定义

### 1.2 当前编译错误
1. **命令行参数解析错误**：多字符的字符字面量（如 `'mt'`）应使用字符串字面量
2. **错误处理问题**：`dyn StdError` 没有实现 `Send + Sync`
3. **临时值生命周期问题**：`format!` 创建的临时值在使用前被释放
4. **部分移动问题**：`representation` 被部分移动后又被借用
5. **生命周期问题**：`i18n/mod.rs` 中需要为 `get` 方法添加显式生命周期
